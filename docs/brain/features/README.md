# Features

## Features mapeadas

- `pje-scraping.md`: scraping de processos via PJE, filas por TRT, login pool, sessoes.
- `pje-documents.md`: extracao de documentos PDF, bookmarks, upload S3.
- `receita-federal.md`: scraping CNPJ e CNDT via Receita Federal.
- `captcha-resolution.md`: resolucao de CAPTCHA (2Captcha, AWS WAF, hCaptcha).

## Quando criar um novo mapa

Use `_template.md` como base quando uma area funcional recorrente nao tiver mapa.

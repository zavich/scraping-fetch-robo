# Runbooks

Guias operacionais para incidentes conhecidos.

## Disponiveis

| Runbook | Severidade | Descricao |
|---------|-----------|-----------|
| `browser-crash.md` | Alta | Browser Puppeteer crashou ou travou |
| `captcha-service-down.md` | Alta | 2Captcha indisponivel ou sem saldo |

## Convencoes

- Use o template `templates/runbook.md` para criar novos runbooks.
- Cada runbook deve ter: Sintoma, Diagnostico, Resolucao, Prevencao.

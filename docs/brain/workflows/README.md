# Workflows

Guias passo-a-passo para tarefas recorrentes.

## Disponiveis

- `investigacao-progressiva.md`: protocolo geral de investigacao.
- `debug-scraping-pje.md`: debug de scraping PJE (timeout, falha, bloqueio).
- `debug-fila-jobs.md`: debug de filas BullMQ (jobs travados, acumulados, falhando).

## Convencoes

- Cada workflow tem secoes: Sintoma, Passos, Resultado esperado.
- Use o template `_template.md` para criar novos workflows.
